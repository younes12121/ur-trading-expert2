## ğŸ› Troubleshooting

### Bot Not Starting?
```bash
# Check if Python processes running
Get-Process python

# Kill all Python processes
taskkill /F /IM python.exe

# Restart bot
python telegram_bot.py
```

### Module Import Errors?
```bash
# Install dependencies
python -m pip install -r requirements.txt
```

### Bot Token Invalid?
1. Check `bot_config.py`
2. Verify token from @BotFather
3. Make sure no extra spaces

### "Conflict: terminated by other getUpdates"?
This means multiple bot instances are running.
```bash
# Kill all Python processes
taskkill /F /IM python.exe

# Restart (only once)
python telegram_bot.py
```
