# BTC Signal Template with Timing Analysis

## Signal Format (with Timing)

Every signal will now include:

### 1. MARKET DATA
- Current price
- 24h stats
- Volume analysis

### 2. TIMING ANALYSIS ‚è∞
- **Entry Timing:** NOW / WAIT / SPECIFIC TIME
- **Market Phase:** Momentum / Consolidation / Reversal
- **Session:** Asian / European / US hours
- **Urgency:** High / Medium / Low
- **Valid Until:** Time when signal expires

### 3. TRADING SIGNAL
- Direction (BUY/SELL/HOLD)
- Entry, SL, TP1, TP2
- Confidence level

### 4. POSITION DETAILS
- Lot size
- Risk/Reward
- Profit potential

### 5. EXECUTION PLAN
- Step-by-step instructions
- Time-sensitive actions

### 6. KEY LEVELS
- Support/Resistance
- Timing for each level

## Timing Categories:

**ENTER NOW** - Immediate entry recommended
- Strong momentum
- High confidence
- Time-sensitive opportunity

**ENTER WITHIN 5-15 MIN** - Entry window
- Good setup forming
- Wait for slight confirmation
- Don't miss the move

**WAIT FOR PULLBACK** - Be patient
- Price extended
- Better entry coming
- Specific level to watch

**WAIT FOR BREAKOUT** - Confirmation needed
- Price consolidating
- Need direction confirmation
- Specific trigger level

**EXPIRED** - Signal no longer valid
- Market conditions changed
- New analysis needed
