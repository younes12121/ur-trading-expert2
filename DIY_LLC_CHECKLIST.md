### **Problem: Can't Get EIN Online**
- Call IRS international line: +1-267-941-1099
- Have LLC documents ready

- Don't panic! ~70% approval for international

- Make sure all info matches LLC documents exactly
- Upload clear passport photo
- Provide business description
- Contact Stripe support if needed

### **Problem: Payment Not Working**
- Verify you're in LIVE mode (not test mode)
- Check API key starts with `sk_live_`
- Verify price IDs are from live products
